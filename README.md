# Jarvis - Assistente Virtual em Python

Projeto de assistente virtual desenvolvido em Python, com foco em reconhecimento de voz, síntese de fala e automação de comandos.

## Visão geral

O Jarvis foi criado como um projeto prático para estudar integração entre voz, processamento de comandos e organização modular em Python. A proposta é construir um assistente capaz de ouvir, interpretar e responder ao usuário.

## Funcionalidades

- Reconhecimento de voz
- Síntese de fala
- Wake word
- Estrutura de memória com JSON
- Testes com integração local de IA

## Tecnologias utilizadas

- Python
- SpeechRecognition
- pyttsx3
- JSON
- Git
- GitHub
